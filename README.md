# Resume_classification
Resume Classification System built using Machine Learning and NLP. The application automatically classifies resumes into job categories such as PeopleSoft, React JS, SQL Developer, and Workday using TF-IDF vectorization and a trained ML model. Developed with Python, Scikit-learn, NLTK, and Streamlit.
# Resume Classification Using Machine Learning & NLP
## Project Overview

This project is a Resume Classification System built using Machine Learning and Natural Language Processing (NLP). The application automatically classifies resumes into different job categories based on resume content.

## Technologies Used

* Python
* Machine Learning
* NLP
* Scikit-learn
* NLTK
* Pandas
* Streamlit

## Features

* Resume text extraction
* Resume preprocessing
* TF-IDF vectorization
* Resume classification
* Skill extraction
* Interactive web interface using Streamlit

## Supported Categories

* PeopleSoft
* React JS
* SQL Developer
* Workday

## How to Run

pip install -r requirements.txt

streamlit run app.py
